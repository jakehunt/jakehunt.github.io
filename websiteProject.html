My Website

I didn't back down and became a better googler because of it. Whenever I thought of some functionality my website should have, I knew that it could be done.

Website Details: This website is hosted on Github Pages and the domain name was acquired on GoDaddy.com. The canvas on the home page is created using processing.js which I learned on KhanAcademy.
